# Computational Geometry: Algorithms and Applications 各版本对比

## 第一版 (1997)

- 首次出版
- 13 章内容
- 作者: Mark de Berg, Marc van Kreveld, Mark Overmars, Otfried Schwarzkopf

## 第二版 (2000)

- 更新部分算法
- 13 章内容
- 增加了更多练习题
- 修正了第一版的一些错误

## 第三版 (2008)

- 15 章内容
- Otfried Schwarzkopf 改名为 Otfried Cheong
- 增加了新的章节和算法
- 更新了参考文献
- 增加了更多实际应用案例

## 第四版 (2023)

- 最新版本
- 重新组织了部分内容
- 更新了算法和应用
- 增加了现代计算几何的新发展
- 加入了更多实际工程案例
